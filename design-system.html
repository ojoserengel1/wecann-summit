Animações

Classes de animação ativadas por scroll (GSAP + ScrollTrigger) ou ao montar o componente. Passe o mouse para re-executar cada animação.

.entradaesquerda

Entrada da esquerda com fade. Parágrafos, imagens e blocos no lado esquerdo.

translateX(-40px) → 0 · opacity 0→1 · 0.7s ease-out
.entradadireita

Entrada da direita com fade. Imagens e elementos no lado direito do viewport.

translateX(40px) → 0 · opacity 0→1 · 0.7s ease-out
.entradacima

Entrada de cima. Headers de seção, badges, elementos que "descem" para o layout.

translateY(-30px) → 0 · opacity 0→1 · 0.7s ease-out
.textosubindo

Texto subindo com blur reveal. Aplicado linha a linha nos títulos grandes (SplitType).

translateY(40px) + blur(3px) → 0 · 0.8s ease-out
.textoopacidade

Fade in simples. Para ícones, separadores e elementos decorativos de suporte.

opacity: 0 → 1 · ease · 1s
.texto3d

Perspectiva 3D + subida. Headings principais para efeito cinematográfico de destaque.

perspective(400px) rotateX(15deg) → 0 · 0.8s ease-out
.textoescala

Scale up com fade. Para cards, logos de patrocinadores e elementos em grid.

scale(0.85) → 1 · opacity 0→1 · 0.6s ease-out
Configuração de timing
--ease-out:
cubic-bezier(0.16, 1, 0.3, 1)
--t-fast: 150ms
--t-base: 240ms
--t-slow: 380ms
